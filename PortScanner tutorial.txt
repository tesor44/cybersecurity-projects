# PortScanner V2

Простой и быстрый сканер портов на Python с использованием многопоточности.

### Функции:
- **TCP Port Scanning**: Использует библиотеку `socket` для проверки статуса портов.
- **Multithreading**: Благодаря `threading` проверяет 500 портов за 2 секунды.
- **Reporting**: Сохраняет результаты в `report.txt` и выводит отчет в консоль.

### Как запустить:
1. Установите Python.
2. Скачайте файл PortScanner V2.py.
3. Запустите через терминал: python PortScanner V2.py.
